## Getting Started

1. Install: `npm install && bower install`
2. Run: `grunt server`
3. Access [http://localhost:9000/](http://localhost:9000/)

## Dependencies

You're going to need:

- [NodeJS](http://nodejs.org/download/)
- [Ruby Compass](http://compass-style.org/install/)
